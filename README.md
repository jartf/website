# Personal Website

This is a personal website, blog, and portfolio built with Next.js, Tailwind CSS, and other modern web technologies. It's designed to be a digital garden, a place for the author to share their thoughts, projects, and experiments.

## Features

-   **Blog**: A collection of articles on various topics, including technology, economics, and personal experiences.
-   **Projects**: A showcase of the author's work, including web development projects, open-source contributions, and other creative endeavors.
-   **Multilingual Support**: The website is available in multiple languages, with a language toggle to switch between them.
-   **Dark Mode**: A dark mode theme that can be enabled or disabled based on user preference.
-   **Responsive Design**: The website is fully responsive and works well on all devices, from mobile phones to desktop computers.

## Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites

-   Node.js (v18 or higher)
-   pnpm

### Installation

1.  Clone the repo
    ```sh
    git clone https://github.com/jarema/new-website.git
    ```
2.  Install NPM packages
    ```sh
    pnpm install
    ```
3.  Start the development server
    ```sh
    pnpm dev
    ```

The website should now be running at `http://localhost:3000`.

## Available Scripts

In the project directory, you can run:

-   `pnpm dev`: Runs the app in the development mode.
-   `pnpm build`: Builds the app for production to the `.next` folder.
-   `pnpm start`: Starts the production server.
-   `pnpm lint`: Lints the code using ESLint.

## Project Structure

The project is organized as follows:

-   `app/`: Contains the main application logic, including pages, layouts, and components.
-   `components/`: Contains reusable UI components.
-   `content/`: Contains the content for the blog and other pages, written in Markdown.
-   `hooks/`: Contains custom React hooks.
-   `i18n/`: Contains the internationalization configuration and translation files.
-   `lib/`: Contains utility functions and other shared code.
-   `public/`: Contains static assets, such as images and fonts.
-   `styles/`: Contains global styles and Tailwind CSS configuration.
-   `translations/`: Contains the translation files for the website.
